# DOM Manipulation
**The TASK :**
+ Create an input box and button. Add tasks to a list when clicked.
+ Add a delete button for each task.
+ Change the background color when a button is clicked.
___
### First:
Creating a div element, in which the to-do list will be displayed.

### Second:
Within the div, an input field and a button called `add task` is added.

### Third:
In Javascript a function `addTask` is created to get the data from the input field and create the list, and `delete buttons` for all the elements of the list dynamically.

### Fourth:
Another function `delTask` is created to delete any task from the list. When user will click the delete button for any task, this function will be performed.
